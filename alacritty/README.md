#Alacritty
##Overview

Alacritty is an Open Source cross platform Terminal Application written with C++ / Rust that utilizes a GPU for rendering.

##Installation

###Mac OS
1. Install Homebrew
2. Brew Install alacritty

##Config file
1. CD .config
2. mkdir alacritty
3. CD alacritty
4. nvim alacritty.yml
